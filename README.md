## Storybook

```sh
npm run watch
```

Starts shadow-cljs in a watcher,compiling the stories namespaces.

In a separate terminal:

```sh
npm run start-storybook
```

starts a webserver.

### Publishing to Github Pages

```sh
npm run deploy-storybook
```
